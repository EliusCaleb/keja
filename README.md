# KEJA
Keja is an app that enable user to view the  hotels and room of that particular  and auto poluutes the information in form for them to see the set date and time for the duration


## Installation

Use the npm install to  react-app packages .
Use bundle install to rails packages

```bash  for rails
 bundle install
 rails db:migrate db:seed
 rails s
```


```bash  for react
  npm install --prefix client
  npm start --prefix client
 
```
